# Checklist manuelle V0.1.1 — ZythoHunt

Ne cocher que les contrôles réellement effectués.

## Carte

- [ ] Vue initiale lisible.
- [ ] Quatre troncs visibles : Ale, Lager, Fermentation spontanée, Fermentation mixte / sauvage.
- [ ] `?scenario=empty` affiche uniquement la structure.
- [ ] `?scenario=cardinal` affiche West Coast IPA sans IPA ni American IPA.
- [ ] `?scenario=secondary` affiche les styles et liens secondaires attendus.
- [ ] `?scenario=mixed` reste lisible.
- [ ] Vue générale inclut les extrémités des liens visibles.
- [ ] Recentrage fonctionnel.
- [ ] Zoom avant fonctionnel.
- [ ] Zoom arrière fonctionnel.

## Souris

- [ ] Pan sans saut brutal.
- [ ] Molette fonctionnelle.
- [ ] Zoom centré sous le pointeur.
- [ ] Aucun déplacement accidentel des nœuds.

## Tactile

- [ ] Pan à un doigt.
- [ ] Pinch à deux doigts.
- [ ] Reprise du pan après pinch.
- [ ] Aucun scroll de page accidentel dans la carte.
- [ ] Scroll normal hors carte.

## Debug

- [ ] `?debug=1` ouvre le panneau.
- [ ] `?scenario=cardinal&debug=1` sélectionne visuellement `cardinal`.
- [ ] Saisie `West Coast IPA`.
- [ ] Saisie `WCIPA`.
- [ ] Saisie `NEIPA`.
- [ ] Saisie invalide.
- [ ] Message « déjà visible ».
- [ ] Masquage.
- [ ] Réinitialisation.
- [ ] Aucune persistance après rechargement.

## Accessibilité

- [ ] Focus clavier visible sur les contrôles.
- [ ] Absence de faux boutons sur les styles.
- [ ] Libellés accessibles.
- [ ] Lisibilité portrait.
- [ ] Lisibilité paysage.

## Navigateurs et appareils

- [ ] Chrome desktop.
- [ ] Firefox desktop si disponible.
- [ ] Safari desktop si disponible.
- [!] iPhone Safari réel : à valider hors environnement agent.
- [ ] Android réel ou émulation.
